## VUCA

- 定义
VUCA是四个单词的缩写，代表了四种状态，它们分别是：Volatile 不稳定、Uncertain 不确定、Complex 复杂、和 Ambiguous 模糊

- 历史
最早在1991年由美军提出，用以描述冷战结束后世界局势呈现的不稳定、不确定、复杂且形式模糊的状态

- 应对
```
Volatility => Vision
Uncertainty => Understanding
Complexity => Clarity
Ambiguity => Agility

```
