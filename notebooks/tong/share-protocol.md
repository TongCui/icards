## 共享访问协议

#### NFS
- 网络文件系统 Network File System
- Sun开发，Oracle收购
- 标准RFC协议

#### SMB/Samba
- Server Message Block
- 最早是微软为自己设计的专用协议，后经逆向重新实现Samba

#### FTP
- File Transfer Protocol
- TCP/IP 应用层对的一个基础协议，通常使用22端口

#### AFP
- Apple Filing Protocol
- macOS
